# Magistrat Vibe Playbook

## Product Voice
- Calm, precise, and editorial.
- Prefer compiler language: finding, evidence, patch, precondition, reconcile.
- Avoid anthropomorphic phrasing.

## Interaction Principles
- Show why a finding exists before suggesting action.
- Show expected impact and reversibility before apply.
- Preserve user agency with explicit review checkpoints.

## Trust Signals
- Deterministic outputs for same input.
- Visible confidence and risk labels.
- Explicit `NOT_ANALYZED` coverage.
- Honest reconciliation after external undo/redo.
